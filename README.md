# Task 2: Data Visualization and Storytelling

## Objective
Create clear and impactful visualizations that communicate a compelling data story.

## Tools Used
- Tableau / Power BI

## Dataset
- Superstore.csv (or any sales dataset)

## Deliverables
- Visual report (PDF) or dashboard screenshots

## Outcome
- Developed strong visual storytelling skills using charts and graphs

## Approach / Key Guidelines
1. Selected appropriate chart types based on data
2. Avoided clutter and excessive use of colors
3. Highlighted key insights and trends
4. Added context and explanations to each chart
5. Focused on business insights rather than visuals alone
6. Created a summary slide / storyboard for clear storytelling
